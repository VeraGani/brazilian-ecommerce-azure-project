# Brazilian E-Commerce Azure Data Warehouse

## Project Overview

This project demonstrates an end-to-end data engineering workflow using the Brazilian E-Commerce Public Dataset (Olist). The project loads raw CSV files into Azure Blob Storage, imports them into Azure Synapse Analytics using PolyBase, performs data transformations with T-SQL, and creates business dashboards for reporting.

## Technologies

- Azure Blob Storage
- Azure Synapse Analytics
- PolyBase
- T-SQL
- Tableau (or Power BI, depending on assignment requirements)
- Python
- Pandas
